Live Interactivity

WallEx includes interactive Dash components: column sorting in the token table updates the graphs in real-time. Built-in responsiveness allows mobile access, though a redesign is planned for better usability on smaller screens (chart stacking, table resizing).

Known Issue (Jan 2025)

In the current version, TP and SL logic may appear active simultaneously under certain conditions. This is under investigation and will be corrected. The app still correctly evaluates the position as a loss or gain based on final state.

Context

This grant application was prepared in one day after discovering the program. Until recently, I underestimated the value of WallEx. Feedback at events and from builders convinced me to bring it forward. This version is a functional MVP, and this grant would help bring it to a stable, public release.

Stack

Backend: Python using APIs from CoinMarketCap, Moralis, Zerion

Frontend: Dash, Plotly, pandas

Other: selenium, dash_ag_grid, dash_daq, JSON (currently) → SQLite (planned)
